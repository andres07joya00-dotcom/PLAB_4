\doxysubsection{Detailed Description}
Declaración de la clase \doxylink{classUsuario}{Usuario}, que representa un cliente con historial de compras. 

La clase {\ttfamily \doxylink{classUsuario}{Usuario}} almacena el nombre del cliente y un historial de compras, donde cada compra se compone de un conjunto (vector) de objetos {\ttfamily itemcarrito}. Permite registrar nuevas compras y visualizar el historial completo.
