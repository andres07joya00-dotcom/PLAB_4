\doxysubsection{Detailed Description}
Declaración de la clase producto, que gestiona la información base de los productos disponibles. 

La clase {\ttfamily producto} almacena listas (vectores) con los nombres, precios y cantidades disponibles de los productos. Sirve como clase base para {\ttfamily itemcarrito}, desde la cual se extienden las funcionalidades relacionadas con el carrito de compras.
