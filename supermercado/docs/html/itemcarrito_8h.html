Detailed Description

Declaración de la clase itemcarrito, que representa un producto dentro del carrito de compras.

Esta clase hereda de la clase producto y agrega información relacionada con la cantidad de productos y el precio total correspondiente a esa cantidad. Además, provee métodos para administrar un vector de ítems en el carrito, como agregar, eliminar y mostrar productos.
